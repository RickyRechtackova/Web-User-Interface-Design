Web Protocols

HTTP
= HyperText Transfer Protocol
- the foundation of data communication
- defines the rules for requesting and transferring web resources (HTML files, images, stylesheets)
HTTPS
= HyperText Transfer Protocol Secure
- secure version of HTTP
- uses encryption to protect the data exchanged
- provides integrity and confidentiality
TCP/IP
= Transmission Control Protocol/Internet Protocol
- TCP - communication between application software (browser) and network software, responsible for breaking down data into IP packets when they're sent and assembling them when recieved
- IP - responsible for sending packets to destination
FTP
= File Transfer Protocol
- enables file transfer from local machine to web server

Web Concepts

URL

= Uniform Resource Locator

- url is a web adress consisting of a protocol, domain name and path.

It can also contain sub-domain.

DNS

= Domain Name System

- translates human readable domain names into IP adresses.

TLD

= Top-Level Domain

- can be generic:

  • .org
  • .com
  • .net

- or country codes:

  • .ie
  • .cz
  • .fr

Cookies

= small pieces of data sent to clients browser

- used for user authentication, tracking user behaviour, remembering site preferences (e.g. location for weather or road information)

Caching

= storing copies of frequently accessed resources

- thanks to that it reduces and improves load times of pages

Browsers

= software apps displaying content using rendering engines (chromium, gecko, webkit)

  • Chrome
  • Safari
  • Opera
  • Microsoft Edge

- they can behave differently, the reasons for that are:

  • They are developed independently
  • They use multiple rendering engines
  • the DOM support varies
  • They each display their own JavaScript

Communication between Client and Server

Request Response Model = client (browser) sends requests and the server delivers data
Stateless Protocol = each request is independent and does not retain previous information, that's where cookies are usually used
Web Sockets = enables two-way communication, used for real time applications like chat, gaming, live updates,...
